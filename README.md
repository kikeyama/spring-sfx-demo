# Java Spring Demo

Compatible with OpenJDK 8, Spring Boot 2.3

## Build on Docker

```
docker build -t <DOCKER_REGISTRY>:<PORT>/<IMAGE_NAME>:<TAG> .
```

example:

```
docker build -t localhost:5000/spring-sfx-demo:latest .
```
